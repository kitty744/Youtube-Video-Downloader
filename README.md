# Youtube Video Downloader
- A very simple pytube application that downloads any youtube video of your choice.
- Just give it a YouTube URL, it will download the video.
- Great if you don't have any internet and you wanna watch some videos, but don't have YouTube premium.
- Instructions below.

# IMPORTANT NOTE
- Before following instructions for running on linux, you must enter the virtual enviroment.
- To do so, open the visual studio code terminal, and type: (source YTDownloader/bin/activate)

# Instrcutions #1 - Windows
--- Requirements ---
1. Visual Studio Code
2. Python and it's libraries (such as pip)
3. pytubefix, download with: pip install pytubefix

After installing all of the things listed above, open a terminal and paste the following command: (python main.py)
This will launch the application!

# Instructions #2 - Linux
--- Requirements ---
1. Visual Studio Code
2. Python3 and it's libraries (such as pip)
3. pytubefix, download with: pip install pytubefix

After installing all of the things listed above, open a terminal and paste the following command: (python3 main.py)
